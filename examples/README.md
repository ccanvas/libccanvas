Here's the recommended order of looking at examples.

1. [basic](https://github.com/ccanvas/libccanvas/tree/master/examples/basic) - basic key event captures
2. [setchar](https://github.com/ccanvas/libccanvas/tree/master/examples/setchar) - drawing on the terminal
3. [snake](https://github.com/ccanvas/libccanvas/tree/master/examples/snake) - building a multi-component app
4. [paint](https://github.com/ccanvas/libccanvas/tree/master/examples/paint) - using multiple spaces
4. [explorer](https://github.com/ccanvas/libccanvas/tree/master/examples/explorer) - using shared variables
