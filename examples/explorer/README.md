# Explorer

This bundle showcases "watching" value updates.

I would recommend looking at all 4 components in the following order:

1. [`explorer-manager`](./explorer-manager)
2. [`explorer-path`](./explorer-path)
3. [`explorer-dir`](./explorer-dir)
4. [`explorer-path`](./explorer-path)

## How to run this component

- [`ccanvas`](https://github.com/siriusmart/ccanvas) installed in system.
- Install all 4 components components using `cargo install --path .`

Start the canvas using the command below.

```sh
ccanvas explorer ccanvas-explorer
```

## Controls

- Up down to move cursor
- Left to go to parent folder
- Right/enter to go to selected folder
- Backspace to go back in history
- Q to exit
